# pokemon-lukmanbayhaqi

## Project setup

### Install packages

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Run sass compiler

- From root folder

```
cd src/styles && sass --watch app.scss style.css
```

- src/styles folder

```
sass --watch app.scss style.css
```

### Compiles and minifies for production

```
npm run build
```
